Conventions
HTML,CSS,Javascript file names are uppercase first letter and rest is lowercase ,if there is a new word first letter is 
uppercase and rest is lowercase

e.g. Login.html , Register.html, ManageAccounts.html

Class: Uppercase then lowercase first e.g. HelloWorld 
ID: START WITH ID , Uppercase then lowercase e.g. ID_Uploader
